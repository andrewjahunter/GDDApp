Church Life

Church Life is a very powerful and flexible system of simple lists that your church/ministry can use to help communicate and group people meaningfully.

All lists are neatly grouped into 2 layers starting with the top level category, followed by the 2nd level fields and then all the list items in that field.

Each list item has three options for you to choose from. Using the navigation tool below simply navigate down to the list you would like edit against your profile and set the required field
